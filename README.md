# 🌌 GEMINI CLONE — Next-Gen AI Assistant









## 🚀 Overview

### Gemini Clone is a beautifully designed, ultra-fast AI assistant powered by Google Gemini 2.0 Flash and hosted on Cloudflare Workers + Pages.


#### It supports:

✨ Text Chat — natural, fast, intelligent

📄 PDF Understanding — summarize, explain, extract

🖼 Image Understanding — describe, analyze, OCR

🎨 AI Image Generation — free via Pollinations

🌗 Dark / Light Mode — with local memory

⚡ Typing Animations & Smooth UI

#### This is NOT just a basic chatbot it's a full AI platform wrapped in a clean, modern UI.
